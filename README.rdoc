= Humanize Number

Rails gem/plugin to describe a number in words.

== Install

Add the following line in to your Gemfile:

  gem 'humanize_numbers'

And from the command line run:

  bundle install

== Usage

This fork has been modified to return English and Spanish, it isn't perfect:

  1235.to_words
  => "one thousand thirty five"
  => ""

  9736.to_words
  => "nine thousand seven hundred thirty six" (English)
  => "nueve mil setecientos treinta y seis" (Spanish)

Enjoy!!!
